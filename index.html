Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Every tag needs to be opened and closed.
  • The information in between the opening and closing tags is called an element.
  • Using HTML gives structure to the webpage.

CSS

  • CSS adds styling to our webpage.
  • A developer can add colors, fonts, and shape to different parts of a webpage using CSS.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript allows a user to interact with a webpage.
